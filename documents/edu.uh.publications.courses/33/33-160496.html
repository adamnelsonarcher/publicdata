COSC 6378 - Network System Programming

Credit Hours: 3.0
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: COSC 6377 and knowledge of C, C++.

Network protocol stacks, socket programming, sensor network programming, network configuration, LAN and WAN measurements using state-of-the-art networking practices.
