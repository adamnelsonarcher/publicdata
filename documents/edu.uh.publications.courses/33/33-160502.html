COSC 6390 - Internet Computing

Credit Hours: 3.0
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: COSC 4330 or consent of instructor.

Introduction to Internet, including http, cgi, server-side programming including object-oriented programming, applets, abstract windowing toolkit, multithreading, event handling, security, network programming and object serialization.
