INDE 4364 - Big Data and Analytics

Credit Hours: 3.0
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: INDE 3310 and 3364.
Description
Fundamental techniques and tools used to design and analyze large volumes of data, data analytics concepts, collection, filtering and classifications using statistical techniques, data mining algorithms and techniques for big data analytics, use of graphical data explorations and visualization.
Additional Fee Y
