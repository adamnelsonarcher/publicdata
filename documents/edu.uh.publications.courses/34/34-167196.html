INDE 2331 - Computer Applications for Industrial Engineers

Credit Hours: 3.0
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: ENGI 1100 and 1331.
Description
Structured programming in Visual Basic with numerical analysis applications in industrial engineering. Introduction to computer aided drafting software.
